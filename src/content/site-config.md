# Site Configuration Guide

This file controls all site-wide settings for your DevFolio portfolio.

## How to Customize

1. **Site Information**: Update `site_name`, `site_title`, `site_description`, and `site_url` with your own information
2. **Author Information**: Change `author_name`, `author_url`, `creator`, and `publisher` to your details
3. **SEO Keywords**: Add keywords relevant to your portfolio and skills
4. **PWA Settings**: Customize the app name and colors for mobile installation
5. **Footer**: Configure footer text and links
6. **Analytics**: Add your Google Analytics or Tag Manager ID to enable tracking
7. **SEO**: Add verification codes if needed

## Important Notes

- All URLs should include the protocol (http:// or https://)
- Colors should be in hex format (#000000)
- Leave analytics fields empty ("") to disable tracking
- The `site_url` should match your deployment URL for proper SEO
